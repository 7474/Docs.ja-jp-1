## <a name="conventions"></a><span data-ttu-id="37978-328">規約</span><span class="sxs-lookup"><span data-stu-id="37978-328">Conventions</span></span>

<span data-ttu-id="37978-329">完全なデータベースを作成できるように Entity Framework の順序で記述する必要があるコードの量は最小限のため*規則*、または Entity Framework は、想定します。</span><span class="sxs-lookup"><span data-stu-id="37978-329">The amount of code you had to write in order for Entity Framework to be able to create a complete database for you is minimal because of *conventions*, or assumptions that Entity Framework makes.</span></span> <span data-ttu-id="37978-330">既に説明した一部の場合、または自分がそれらを認識せずに使用されていたは。</span><span class="sxs-lookup"><span data-stu-id="37978-330">Some of them have already been noted or were used without your being aware of them:</span></span>

- <span data-ttu-id="37978-331">エンティティ クラス名の複数形のフォームは、テーブル名として使用されます。</span><span class="sxs-lookup"><span data-stu-id="37978-331">The pluralized forms of entity class names are used as table names.</span></span>
